complete project description

This project allows the user to control a small robot through commands sent via sms or email. The project uses an arduino microcontroller to control the robot's motors. Sms commands are processed using Twilio and email commands are processed using mailjet. Commands are sent to the arduino over serial from python. The entire robot was custom designed in cad and 3d printed in pla plastic.
